# Advanced Statistical Analysis (2021)

## 📌 Project Overview
This project demonstrates the application of multivariate statistical techniques to solve complex business and research problems. It covers experimental design analysis and dimensionality reduction.

## 📊 Key Problems Solved
1. **Hay Fever Relief Analysis (Experimental Design):** - Performed a **Two-Way ANOVA** to study the effect of two active ingredients (A & B) at three different levels each.
   - Analyzed interaction effects to determine the most effective combination for symptom relief.
2. **University Dimensionality Reduction (PCA):**
   - Applied **Principal Component Analysis (PCA)** to a dataset of 17 variables describing various colleges.
   - Reduced the feature space while retaining maximum variance to identify the most significant factors in comparing educational institutions.

## 🛠️ Technical Skills Applied
* **Analysis of Variance (ANOVA):** Testing for main effects and interaction effects between independent variables.
* **PCA (Principal Component Analysis):** Calculating Eigenvalues, Eigenvectors, and Cumulative Variance to reduce data dimensionality.
* **Statistical Visualization:** Using interaction plots and scree plots to interpret complex data relationships.
